# Keyboard Shortcuts

Complete reference of keyboard shortcuts in Scamp.

## Tools

| Shortcut | Action |
|---|---|
| **V** | Select tool |
| **R** | Rectangle tool |
| **T** | Text tool |

## Actions

| Shortcut | Action |
|---|---|
| **Cmd+D** | Duplicate selected element |
| **Delete** / **Backspace** | Delete selected element |
| **Cmd+G** | Group selected elements |
| **Cmd+Z** | Undo |
| **Cmd+Shift+Z** | Redo |
| **Cmd+S** | Save (commits CSS editor changes) |

## Zoom

| Shortcut | Action |
|---|---|
| **Cmd+=** | Zoom in |
| **Cmd+-** | Zoom out |
| **Cmd+0** | Reset zoom to 100% |

## Panels

| Shortcut | Action |
|---|---|
| **Ctrl+`** | Toggle built-in terminal |

## Notes

- On Windows/Linux, replace **Cmd** with **Ctrl**.
- Tool shortcuts (V, R, T) only activate when you are not focused on a text input or the CSS editor.
- See individual feature pages for context on each shortcut: [Canvas](canvas.md), [Undo/Redo](undo-redo.md), [Terminal](terminal.md).
